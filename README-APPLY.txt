Cintex UI Refresh Pack (Light, clean, higher contrast)

How to apply:
1) Unzip this package
2) Copy each file into your project, overwriting the existing file at the same path:
   - app/globals.css
   - tailwind.config.ts
   - components/Sidebar.tsx
   - components/Topbar.tsx
   - app/app/layout.tsx
   - app/login/ui.tsx

Then restart dev server:
  npm run dev -- -p 3001
